# [16,21,11,8,12,22] -> Merge Sort
## **1. Yukarı verilen dizinin sort türüne göre aşamalarını yazınız.**

* **input [16, 21, 11, 8, 12, 22]**

* **1**
* [16,21,11]
* [16] [21,11]
* [16] [21] [11]
* [16] [11,21]	
* [16,11,21]

* **2**
* [8,12,22]
* [8,12] [22]
* [8] [12] [22]
* [8,12] [22]	
* [8,12,22]

* **output: [8,11,12,16,21,22]**

## 2. **Big-O gösterimini yazınız.**
* O(nlogn)
